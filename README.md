# Joseph Pagliuca's Portfolio Website

This is a [Next.js](https://nextjs.org) project built with [Tailwindcss](https://tailwindcss.com/). It showcases 
my skills with various different programming langauges and frameworks, along with projects I have worked on as
a software developer and Software Enginerring student.

# Deployment

The website is deployed on [neocities.org](https://neocities.org/).

Deployment is done through the CI/CD pipeline. To keep the website up to date, whenever a branch is merged
to main, the pipepline will build the project and send the HTML, CSS and JavaScript files to my neocities
website and update it in real time.
